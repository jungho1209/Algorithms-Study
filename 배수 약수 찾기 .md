# 배수와 약수

## 배수 , 약수

입력 받은 수가 나누었을 때 나머지 값이 0이되면

약수 혹은 배수인 점을 활용해서 풀음

```c
#include <stdio.h>
int main() {
	int a, b;
	while (1) {
		scanf("%d %d", &a,  &b);
		if (a != 0 && b != 0) {
			if (b % a == 0)
				printf("factor\n");
			else if (a % b == 0)
				printf("multiple\n");
			else
				printf("neither\n");
		}
		else
			break;
	}
}
```
